# Aout Edu — Simulasi SKD CPNS 2026

Web statis siap GitHub Pages, Vercel, atau Netlify.

## Fitur
- Kode akses `aoutedu335`
- 110 soal dari paket PDF Aout Edu
- Timer 100 menit, auto-finish, dan penyimpanan progres lokal
- Stimulus Gambar A untuk 31–36 dan Gambar B untuk 37–40
- Skor TWK, TIU, TKP, dan total
- Download PDF jawaban peserta tanpa kunci/penanda benar-salah
- Pesan WhatsApp otomatis ke Kak Affie
- Responsif HP, tablet, dan desktop

## Deploy Vercel
1. Ekstrak ZIP.
2. Upload seluruh isi folder ke repository GitHub.
3. Import repository di Vercel.
4. Framework preset: **Other**. Tidak perlu build command.
